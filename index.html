📊 Proyectos y Dashboards

Análisis estadístico e inferencial de COVID-19

Exploración de datos de salud en Veracruz (2023), enfocada en la relación entre COVID-19 y tres factores de riesgo: obesidad, diabetes y tabaquismo. Los datos, obtenidos de Datos Abiertos de la Dirección General de Epidemiología, fueron limpiados y procesados con R antes de realizar análisis estadísticos descriptivos e inferenciales para identificar patrones relevantes.

Predicción de Combate Pokémon con Machine Learning

Este proyecto utiliza técnicas de Machine Learning para predecir el ganador de un combate entre dos Pokémon, combinando datos estadísticos y combates históricos con una interfaz web interactiva desarrollada con Flask.
